# 5-Day AI Agents: Intensive Vibe Coding Course 🤖✨

Welcome to my repository dedicated to the **5-Day AI Agents Intensive**, hosted by Google and Kaggle. This course explores the massive paradigm shift in the Software Development Life Cycle (SDLC)—moving from traditional syntax writing to **"Vibe Coding"** (intent-driven programming) and acting as a system orchestrator.

Here, I document my journey designing the evaluation, constraint, and context harnesses for autonomous AI agents.

---

## 🚀 Course Overview & Paradigm Shift
Traditional engineering is evolving. Throughout this intensive, the focus shifts from manual line-by-line coding to guiding autonomous agents that execute complex tasks. As a software engineer, my role here is to orchestrate:
* **Intent & Context:** Defining clear goals and boundaries for the AI.
* **Evaluation Harnesses:** Testing and ensuring agent reliability.
* **Tools & Constraints:** Guardrailing autonomous agents for production readiness.
